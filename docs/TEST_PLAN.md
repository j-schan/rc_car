# 기능 테스트 계획서 (Functional Test Plan) — Phase 1

- 프로젝트: `rc_car` / 대상: 점퍼선 프로토타입 Rev.A
- 원칙: **전원 → 신호 → 통신 검증 후에 모터 연결** (중요 기능 테스트 후 구동 테스트)

---

## T0. 배선 점검 (전원 인가 전) — 목시·도통

| # | 항목 | 방법 | 합격 기준 | 결과 |
|---|------|------|-----------|:---:|
| T0-1 | ENA/ENB 점퍼캡 제거 | 목시 | 캡 2개 제거됨 | ☐ |
| T0-2 | HC-05 TX/RX 엇갈림 | 목시 | PA9→RXD, PA10←TXD | ☐ |
| T0-3 | 공통 GND | 도통(멀티미터) | 3S−·2S−·L298N·서보·HC-05·보드 GND 모두 도통 | ☐ |
| T0-4 | 전원 +극 분리 | 도통 | 11.1V(+)와 7.4V(+) 간 절연 | ☐ |
| T0-5 | 서보 V+ 경로 | 목시 | 서보 빨간선 → 2S(7.4V)에만 | ☐ |

## T1. 전원 검증 (모터·서보 미연결, 무부하)

| # | 항목 | 방법 | 합격 기준 | 결과 |
|---|------|------|-----------|:---:|
| T1-1 | 3S 팩 전압 | 멀티미터 | 11.1~12.6V | ☐ |
| T1-2 | 2S 팩 전압 | 멀티미터 | 7.4~8.4V | ☐ |
| T1-3 | L298N 5V 출력 | 멀티미터 | 4.8~5.2V | ☐ |
| T1-4 | 보드 3V3 | 멀티미터 | 3.2~3.4V | ☐ |

## T2. 신호 검증 (로직 애널라이저 권장 / 없으면 LED·서보로 대체)

| # | 항목 | 방법 | 합격 기준 | 결과 |
|---|------|------|-----------|:---:|
| T2-1 | 모터 PWM 주파수 | PA6 파형 | 1kHz ±1% | ☐ |
| T2-2 | 모터 PWM 듀티 | '5' 명령 후 PA6 | ≈55% (555/999) | ☐ |
| T2-3 | 서보 PWM | PA0 파형 | 50Hz, 펄스 1500µs(중립) | ☐ |
| T2-4 | 방향 GPIO | F 명령 후 PB0/PB1 | IN1=H, IN2=L | ☐ |

## T3. 통신 검증 (USB만, 배터리 불필요)

| # | 항목 | 방법 | 합격 기준 | 결과 |
|---|------|------|-----------|:---:|
| T3-1 | HC-05 페어링 | 폰 BT 설정 (PIN 1234/0000) | 페어링 성공, LED 점멸 변화 | ☐ |
| T3-2 | BT→보드 수신 | 앱에서 'F' 전송 | USART2 터미널에 'F' echo | ☐ |
| T3-3 | 연속 명령 | F,B,L,R,S,0~9 연타 | 모두 echo, 멈춤 없음 | ☐ |

## T4. 구동 테스트 (★차를 받침대에 — 바퀴 공중)

| # | 항목 | 방법 | 합격 기준 | 결과 |
|---|------|------|-----------|:---:|
| T4-1 | 서보 중립 | 2S 연결, 부팅 | 앞바퀴 정면 정렬 | ☐ |
| T4-2 | 조향 L/R | 'L','R' 전송 | 좌/우로 부드럽게, 떨림 없음 | ☐ |
| T4-3 | 전진 | 'F' | 뒷바퀴 2개 모두 전진 방향 회전 | ☐ |
| T4-4 | 후진 | 'B' | 반대 회전 | ☐ |
| T4-5 | 정지 | 'S' | 즉시 정지 + 서보 중립 | ☐ |
| T4-6 | 속도 단계 | '3'→F, '9'→F | 회전 속도 차이 육안 확인 | ☐ |
| T4-7 | 좌우 방향 일치 | F 상태 관찰 | 두 바퀴 같은 방향 (반대면 IN3/IN4 스왑) | ☐ |

## T5. 통합(실차) 테스트 — 바닥 주행

| # | 항목 | 합격 기준 | 결과 |
|---|------|-----------|:---:|
| T5-1 | 직진성 | F에서 큰 쏠림 없음 (쏠리면 서보 중립값 보정) | ☐ |
| T5-2 | 조향 반경 | L/R 회전 정상 (~0.55m 반경) | ☐ |
| T5-3 | 최대속도 주행 중 조향 | 서보 떨림/보드 리셋 없음 | ☐ |
| T5-4 | BT 거리 | ≥5m에서 조종 가능 | ☐ |
| T5-5 | 연속 주행 | 5분 주행 후 L298N 발열 손댈 수 있는 수준 | ☐ |
